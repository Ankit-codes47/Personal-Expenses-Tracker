# 💰 Personal Expense Tracker

A fully functional personal finance management web application designed to help me record income and expenses, monitor spending, manage monthly budgets, track savings goals, and understand my financial activity.

The application is being built as a responsive web application and will later be converted into an installable Progressive Web App (PWA) for both desktop and mobile use.

---

## 🎯 Project Goal

The goal of this project is to create a private and secure personal expense management system that allows me to manage my financial data from one place.

This is not a prototype. The final application will use real authentication, cloud data storage, dynamic calculations, financial reports, and cross-device access.

---
